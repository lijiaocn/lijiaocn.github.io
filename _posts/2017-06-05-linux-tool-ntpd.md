## 安装ntp

	yum install ntp ntpdate ntp-doc
	chkconfig ntpd on

## 手动同步一次时间

	ntpdate pool.ntp.org

## 启动ntpd

	systemctl start ntpd

ntpd服务会自动与/etc/ntp.conf中配置的ntp server通信，保证系统时间与ntp server的时间一致。
