## MacOS

	brew install dnsmasq
	cp /usr/local/opt/dnsmasq/dnsmasq.conf.example /usr/local/etc/dnsmasq.conf
	sudo brew services start dnsmasq

## CentOS

## 配置cname

dnsmasq只支持为/etc/hosts中或者通过DHCP获得hostname做cname。

在/etc/dnsmasq.conf中添加配置:

	cname=

重启dnsmasq。
