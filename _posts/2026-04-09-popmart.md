最近泡泡玛特的股价高位腰斩引发关注，段永平的一番点评又将热度向上推了一个量级。不能免除「大佬就是对」的俗气，赶紧把《因为独特》找来阅读。读完以后感慨颇多，解除了很多疑惑。

人们（包括我）对泡泡玛特的质疑主要有三个：1、泡泡玛特的营收是不是主要依靠盲盒能开出稀有款营造出的非理性的消费？2、labubu一个 IP 撑起了38%的营收，如果labubu不火了怎么办？3、泡泡玛特能持续创造出热门 IP 么？

读完《因为独特》，又仔细看了盲盒商品，对这些问题有了一点理解。

## 营收是靠非理性消费支撑的么？

先说盲盒。起初我以为泡泡玛特主要就是靠盲盒炒作支撑营收，毕竟当年他们请的郭麒麟都调侃来到了境内最大的博彩公司。王宁在书中讲到盲盒只是一种销售方式，和打折一个性质。看了下在售的盲盒商品，理解了这个说法。

泡泡玛特的盲盒是按系列划分的，一个系列通常是10个左右造型，其中一个是隐藏款。每一个IP都会持续增加新系列，每个系列中都有一个隐藏款。泡泡玛特没有刻意制造极度的稀缺，因为随着系列的增加，隐藏款的种类是在同步增加的（占总款式的1/10），稀缺性会被稀释。

如果要制造极度的稀缺，应该按照整个 IP 抽盒，整个盲盒池每月一轮换，但是稀有款始终只有那么几个，形成类似彩票大奖那样的稀缺性。激励人们每个月都购买一次，一旦开出稀有款，就如同中几百万大奖。

泡泡玛特没有这样做或者不能这么做，而是选择按系列抽盒。

在指定的系列中抽盒，盲盒是在用户看好某个系列之后增加了一点点随机性。这种方式下，我相信应该是理性消费占主导，为了避免用户失望，系列中每一款还都要做成用户喜欢的样子。盲盒的主要贡献显然只是刺激预算充足的客户一次购买多个，比如「端盒」。

所以对于第一个问题，我的认为泡泡玛特的营收不是依靠制造稀缺营造的非理性的消费，而是依靠潮玩设计激发用户喜爱产生的理性消费，它的营收有真实需求支撑。（泡泡玛特本身也是发现这种需求的真实性后，才主做潮玩）。

## labubu 能持续贡献高营收么？

再说 labubu 这个热门 IP 的问题。labubu 在2023年营收 3.68 亿，2024年出圈，2025 年营收达到 141 亿，占据公司总营收的 38%。这一个 IP 的营收波动就会公司整体产生巨大影响，它的营收能保持稳定么？

很多文章里已经举例证明过 IP 生命周期的问题。IP 生命周期可以非常非常长，hello kitty 历经 50 年依然出现在各类商品上，米老鼠历经百年依然被人熟知。不过拍脑袋就能想到，昙花一现的例子应该会有更多。

一个 IP 能存活多久，不能单靠 IP 自身素质，维护的贡献更大。好比一个石头房子，如果从来不修缮，再坚固也会被时间长河淹没，如果一直维护修缮，理论上可以无限期存在。王宁在访谈中表达了自己的认识，泡泡玛特显然不会让 IP 自生自灭，而是会持续投入资源让 IP 停留在各个年龄段的人们的认知中。

IP 能长久存活不代表贡献的营收稳定。labubu 是在 2024 年引发全球潮流的情况下，在 2025 年贡献了 141 亿的营收。未来 10 年还能维持在这个营收水平么？会不会几年后断崖下跌，回到出圈前几个亿的水平？

全球范围内对 labubu 潮玩的稳定需求是多少，应该很难算出来，所以无法推算 labubu 未来长期稳定的营收会更高还是更低。但可以确定的是，泡泡玛特有办法拓展这个 IP 的营收。

第一个办法是不停地凹造型。labubu 如果始终只有一个姿态，喜欢的人买一个以后消费就结束了，不喜欢的人也永远不会消费，营收很快会下滑到和新出生人口数量挂钩的水平。

但 labubu 不是静态的图案，而是人形的，能够与时俱进地摆出各种各样的姿态、搭配各种各样新鲜潮流的装饰。新的姿态、新的样式，会不停地给人们带来新鲜感，引发老顾客的复购以及戳中新的人群。

第二个办法是拓展品类。labubu 如果只有搪胶材质的摆件，即使不停地凹造型，营收贡献也将局限于潮玩摆件。如果将品类扩展一下，营收范围就可以扩大。比如材质从搪胶拓展到毛绒，用途从摆件拓展到抱枕、挂件、饰品，形象从在玩具上出现延伸到在乐园、衣物、各种日用品上出现等。

每拓展一个品类，就可以增加新的营收。潮玩之外的品类，不需要自有工厂，只需要代工、形象授权、以及和优质品牌联名等。更关键的是，拓展出来的品类会在日常使用中逐渐消耗，产生源源不断的购买需求。在消耗更快的日用品中上出现，还能强化人们对 IP 的认知。 

所以针对第二个问题，我的理解是：可以通过合理的投入维持新老人口对IP 的认知，使 IP 跨越数十年的时间，然后通过不停地凹造型和拓展品类，将一个 IP 的营收维持在超出潮玩范畴的高位上。

凹造型和全品类带来的营收增长能否抵消 labubu 潮流消退引发的下滑？这个问题依然没法回答。但是有乐观的因素可以在一定程度上抵消对潮流消退的顾虑：labubu 距离拓展到全品类还早得很，其它贡献了 62% 营收 的 IP，也能不停凹造型和戳中新的人群，也具备全品类拓展的潜力，全球化从2019年开始才刚 7 年（中间有3年是无法正常经营的口罩期）。

泡泡玛特有一个贡献了 38% 营收的超级 IP，这首先是一件好事，风险只在于这件好事带来的估值有没有过高。我觉的潜力还有，可以相信，更应该关心的是 labubu 有没有被管理层过分透支，会不会烂大街。考虑到王宁对线下店铺呼吸感等方面的苛刻要求，以及需要艺术家参与决策的机制，烂大街的概率很低。

## 泡泡玛特能持续创造热门IP么？

王宁自己在访谈中讲过、通过阅读也感受到，泡泡玛特不是一个静态的、将手脚束缚起来的公司，而是一个在持续地进化的组织。

通过王宁以及他的同事的讲述，可以确信泡泡玛特不是含着金钥匙出生的精英创业，也不是诞生于天才的灵光闪现。而是在日复一日的进货卖货经营中，逐步找到好卖的商品，逐步把店铺改造成更有助于卖货的形态，在持续的进化中抓住了潮玩这个品类。

王宁家庭背景普通，出生于河南农村，父母在县城做小买卖。学习成绩应该非常普通，2005 年考入郑州大学西亚斯国际学院，在当时应该是三本院校。起点比王宁高的人很多很多。

他实际从 2005 年就开始做生意，大一成立社团在学校售卖5元/8元一张入学/毕业纪录光盘。大二（2006年）在学校门口的商业街租了一个店铺开设格子街，开始零售小生意。在北京成立泡泡玛特时，已经积累了 4 年开店零售经验、20万的本钱、和一个能承担店面工作的校友团队。

他起初应该没有给公司一个特别清晰的定位，应该也没有想过开天辟地的宏大计划，只是一个同事听了他在学校开店的事情后评论「如果在北京开一定能更好」。于是继续做在学校里就开始的小店生意，依靠的是在学校期间积攒的本钱以及校友团队。

刚开始经营困难需要信用卡周转，成立一年后「侥幸」得到天使投资人200万资金，维持4年。2014年底发现代理的 Sonny Angel 销售额最好，开始逐步聚焦潮流玩具销售。2015年被 Sonny Angel 版权方牵制，以及在耐克创始人自传《鞋狗》的启发下，主动寻找潮玩 IP 逐步形成泡泡玛特品牌。

王宁从大一开始的十多年经历，本质上就是不停地从小的生意转换到大一点的生意，从一般的模式转换到好一点的模式。从一年需要制作两次的光盘纪录片，转换到效率更高的一手进货一手出货的零售，逐步淘汰不好卖的商品，增加好卖的商品，然后主营潮玩、自营潮玩、IP 运营。

今天的泡泡玛特是从2006年开始经过19年的进化才形成的。有运气的成分，比如天使资金支持、代理 Sonny Angel，但更多依靠的还是创始人王宁的商业嗅觉和进化。比如他特别关注店铺的设计和商品陈设，在2014年遇到撑起 30% 营业额的 Sonny Angel 之前，泡泡玛特店铺和商品互相驱动的进化已经持续了 5年，而王宁的零售、逛街看店经验积攒了 9 年。

现在，泡泡玛特的进化还在进行中。正在从潮流玩具转成 IP 运营（围绕IP的集团化），正在拓展能够使用这些IP的其它生意，比如授权、比如乐园，从潮玩品牌直营转换成商业空间更广阔的 IP 运营。

所以我的观点是：

泡泡玛特能否持续创造出热门的 IP 很难确定，但是王宁大概率会驱动泡泡玛特进化成一个赚钱效率更高的组织。他过去19年的经历以及在访谈中展现出的认识可以为此背书。有「尊重时间、尊重经营」的清醒认识和线下直觉，至少泡泡玛特乐园成功的可能性非常大。如果现在这些IP都不火了，那时候王宁可能已经带领公司进入了一个更大的市场。

当然了，如果把希望寄托在王宁给公司找到了新的方向，那就纯粹是风险投资的范畴。退而求其次，如果泡泡玛特不具备量产热门IP的能力，那么热门 IP 可能就不存在量产的方法，现有的因为各种机缘巧合而形成的 IP 就更值钱了。有 10 个甚至几个长青 IP 就足够了。

《因为独特》这本书非常棒，买本纸质版长期收藏。

## 王宁卖星星棒的故事

王宁在《因为独特》中讲了自己卖星星棒的经历。

2007年底，王宁正在读大三，寒假和团队去义乌看到一个进价只有几毛钱的星星棒，感觉这个商品可以在情人节和鲜花店的鲜花一起销售，当即注册公司订货。后来发生雪灾，货物延期送达，积压了很多星星棒。他们去商场门口摆地摊，摊前立马人山人海，卖得很好。

从这个故事里可以看出很多内容：

第一，大三的时候，王宁就已经有了一个小团队，知道去义乌进货，能够注册公司，即使放在现在来看，也非常早熟；

第二，在寒假的时候预想到星星棒可以在情人节时放在花店里销售，摆地摊销售也很火爆，对商品的嗅觉是对的；

第三，王宁当时只是一个学生团队，压在手里的星星棒应该占用了大量本钱，面对困难没有放弃，去商场门口摆地摊盘活了。

泡泡玛特成立一年后得到天使投资人200万投入，靠这笔钱维持4年，2014年获得了2000万的A轮投资。那是泡泡玛特还没有全力转向潮玩（2015年代理的Sonny Angel爆火，2016年才签约MOLLY）。如果经营上没有任何亮点，是不太可能得到这些投资的，甚至已经倒闭。

这个经历应该也给王宁留下非常深刻的认识：好产品才是第一位的。
